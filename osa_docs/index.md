# LIR Agent

## Overview
The LIR Agent is a Python-based tool designed to facilitate interaction with an AI agent through a command-line interface. It provides functionalities to manage and utilize Python tools within a virtual environment, enabling dynamic execution and registration of these tools.

## Purpose
The primary purpose of the LIR Agent is to offer a seamless interface for users to engage with an AI agent, allowing for real-time streaming of the agent's thoughts and responses based on user input. It supports the creation and management of Python tools, enhancing the flexibility and extensibility of the AI interaction experience.
